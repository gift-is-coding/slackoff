## 键盘快捷键（当前实现）

| 键 | 功能 |
|---|---|
| `W` / `↑` | 上移一条 |
| `S` / `↓` | 下移一条 |
| `Y` / `Enter` | 确认当前步骤（plan → exec → done）|
| `I` | 忽略当前条目（直接从列表移除）|
| `/` | 激活指令输入，设为 AI指示 模式 |
| `:` | 激活指令输入，设为 直接回复 模式 |
| `F` | 切换 P0/P1 过滤 |
| `←` / `→` 或 `⌘1` / `⌘2` | 切换 待处理 / 已处理 tab |
| `?` | 快捷键帮助面板 |
| `Esc` | 退出焦点 / 关闭帮助 |

## Slash 输入两种模式

```
按 /  →  [/] AI指示 模式
         输入框内容：/rewrite 更简洁
         openclaw 收到：Operator instruction: rewrite 更简洁

按 :  →  [:] 直接回复 模式
         输入框内容：:好的，我周末有空
         openclaw 收到：reply with EXACTLY: 好的，我周末有空
```

## 数据来源

```
notification-inbox.json (OpenClaw 写入)
  ├── id, content, source, app, time, status
  └── ai_reply  ←── 显示在 "// AI 建议" 区域（OpenClaw 生成）

Slackoff 决策写回：
  ├── notification-inbox.json status 字段（approved / ignored / executed）
  └── bridge/inbox/*.json（供 OpenClaw 消费的决策信封）
```
